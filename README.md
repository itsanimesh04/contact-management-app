A simple contact management application where you can add, view, and delete contacts.

## What Does This App Do?
- Add new contacts with Name, Email, Phone, and Message
- View all your contacts in a list
- Delete contacts you don't need
- All data is saved in database
- Clean black and white design

## Technologies Used
- **Frontend:** React (with Vite)
- **Backend:** Node.js and Express
- **Database:** MongoDB

## Features
- Form validation (all fields required, email must be valid)
- Success messages when you add or delete contacts
- Contacts are saved in MongoDB database
- Responsive design - works on mobile and desktop
- No page reload needed - everything updates instantly
